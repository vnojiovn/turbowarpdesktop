# turbowarpdesktop
TurboWarp Desktop thay thế Scratch 3 Desktop
- Cải thiện hộp thoại xác nhận khi thoát ứng dụng.
- Thay nhân vật mặc định bằng **Mèo Scratch**.
- Phát hành gói cài đặt **.deb** cho Ubuntu và các bản phân phối Debian.
# Lệnh tải về cài
- wget https://github.com/vnojiovn/turbowarpdesktop/releases/latest/download/TurboWarp-linux-amd64-1.16.0.deb
- sudo apt install ./TurboWarp-linux-amd64-1.16.0.deb
